To-Do Task Managament Application

A simple To-Do App built using Laravel + Vue.js + MySQL

##Features: 

- Add, delete, update tasks

- Mark tasks as complete

- Responsive UI


1. Clone repo:

    git clone https://github.com/your-username/todo-application.git

    cd todo-application

2. Frontend:

    cd frontend

    npm install

    npm run dev

3. Backend:

    cd ../backend

    npm install

    cp .env.example .env

    npm run migrate    
 
    npm run seed   
     
    npm run dev
